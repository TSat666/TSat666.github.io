Présentation de l'entreprise Soft&Co

Soft&Co

Soft&Co est une PME d’édition de logiciels applicatifs située à Vienne. C’est une SASU travaillant notamment avec des acteurs du transport de marchandises tels qu’Airbus. J’y ai réalisé mes deux stages de BTS SIO option SLAM :

  • 1er stage : mai à juin 2024
  • 2e stage : janvier à février 2025

Composition de l'équipe

  • 2 Développeurs
  • Responsable Informatique
  • Assistant Technique
  • Assistant Commercial
  • Directeur Technique
  • Chef Produit

Méthodologie de l'entreprise

Soft&Co utilise la méthodologie Agile SCRUM. Les projets sont découpés en sprints de deux semaines comprenant :

  • Réunion de planification
  • Réunions quotidiennes (daily meetings)
  • Revue de sprint
  • Rétrospective pour améliorer le processus

Technologies utilisées

L’entreprise utilise :

  • WinDev
  • Visual Studio Code
  • Docker
  • ApiDog
  • PostgreSQL
  • Windows 11 sur les postes

Soft&Co propose deux versions de son logiciel :

  • Client lourd WinDev installé sur poste ou serveur
  • Client léger Web SAAS accessible via navigateur

Organisation et méthodes

Les projets sont gérés via GitHub et Jira. L’entreprise applique :

  • Convention CamelCase
  • Tests avant chaque commit
  • Organisation stricte des fichiers
  • Respect de la RGPD et des normes du transport

Ce que j'ai appris

Au-delà des compétences techniques, j’ai appris à :

  • m’intégrer dans un environnement professionnel
  • collaborer efficacement en équipe
  • présenter mes travaux à l’oral
