Présentation de mes Projets

Projet du Stage de Première Année

Objectif du projet

Durant mon stage de Première Année, on m'a donné 4 semaines pour ajouter une section Signataire permettant de gérer les signatures sur les documents PDF générés sur la version Web SAAS du logiciel nommé DMDLog.

Présentation du logiciel

DMDLog est un logiciel métier qui permet de traiter, gérer et éditer les documents de transport de marchandises dangeureuses.

Organisation du travail

J’ai participé aux réunions d’ouverture et de fermeture de sprint avec mon maître de stage, le scrum master et le directeur commercial. Le livrable était le code validé du ticket attribué, puis fusionné sur GitHub.

Conception et développement

Développement Back-End

Avec mes collègues, nous avons créé la table Signataire dans la base de données via Prisma, ce qui m’a permis de développer la partie Back (API + BDD).

Intégration finale

Enfin, j’ai intégré les signataires aux autres modules pour permettre l’ajout automatique des signatures PDF.

Gourmetise

Projet from scratch : avec un camarade, j’ai créé un site web Vue.js permettant de s’inscrire à un concours de boulangerie via une API Symfony connectée à une base HeidiSQL.

J’ai également développé une application mobile utilisant une base SQLite, permettant d’évaluer les boulangeries et d’envoyer les résultats vers la base HeidiSQL.

À la fin du concours, l’administrateur peut générer les résultats (top 3 des boulangeries) grâce aux notes envoyées par l’application mobile.
