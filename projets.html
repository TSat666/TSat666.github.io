Présentation de mes Projets

Projet du Stage de Première Année

Durant mon stage de Première Année, on m'a donné 4 semaines pour ajouter une section Signataire permettant de gérer les signatures sur les documents PDF générés sur la version Web SAAS du logiciel.

J’ai participé aux réunions d’ouverture et de fermeture de sprint avec mon maître de stage, le scrum master et le directeur commercial. Le livrable était le code validant le ticket attribué, fusionné sur GitHub.

J’ai commencé par réaliser un schéma de l’onglet Signataire, puis j’ai participé à la création des SPECS. Ensuite, j’ai développé le front-end. Voici quelques images :

Avec mes collègues, nous avons créé la table Signataire dans la base de données via Prisma, ce qui m’a permis de développer la partie Back (API + BDD).

Enfin, j’ai intégré les signataires aux autres modules pour permettre l’ajout automatique des signatures dans les PDF générés.
